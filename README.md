# Setup
## Install environments
```shell
$ python -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```
## Start jupyterlab and mlflow
```shell
$ jupyterlab --port 9999 $
$ mlflow server --host 127.0.0.1 --port 8080 $
```
